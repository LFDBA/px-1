Synth Device Documentation

Using the device menus

This page documents the current menu behavior of the device: what each screen does, what Knob 1 through Knob 4 control, and how the Edit button behaves.

Main menu: turn any knob to move Open: press the Edit button once Back: double-click the Edit button

Basics

Start here if you only need the overall flow before diving into the individual menus.

Main navigation

From the main menu, turn any knob to move the highlight. Press the Edit button once to open the highlighted menu.

Editing sound menus

Tone, Wave, ADSR, Reverb, Noise, and Harmonist only change values while editing is active. Press the Edit button once to start editing, then press it again to stop.

Special cases

Write and Presets use the Edit button differently than the other menus. Their full behavior is listed in their own sections below.

Menu reference

The main menu contains eight destinations: Tone, Wave, ADSR, Reverb, Noise, Harmonist, Write, and Presets.

Main Menu

Main

The launcher for every other screen.

  • Knob 1-4: Turning any knob moves the selection up or down the menu list.
  • Edit button: Opens the highlighted menu.
Tone

Tone

Core output controls plus the live waveform history display.

  • Knob 1: Output level.
  • Knob 2: Octave.
  • Knob 3: Waveform history length shown on the display.
  • Knob 4: Clip amount for added saturation.
  • Edit button: Press once to start or stop editing.
Wave

Wave

Edit a custom waveform or blend through the built-in waveform shapes.

  • Knob 1: Selects which waveform point you are editing.
  • Knob 2: Moves the selected point up or down.
  • Knob 3: Changes the curvature or smoothing of the custom wave.
  • Knob 4: Morphs through the built-in waveforms. Using it returns the screen to the built-in morph wave instead of the custom one.
  • Edit button: Press once to start or stop editing.
ADSR

ADSR

Envelope timing and sustain shape.

  • Knob 1: Attack.
  • Knob 2: Decay.
  • Knob 3: Sustain.
  • Knob 4: Release.
  • Edit button: Press once to start or stop editing.
Reverb

Reverb

Room and decay controls for the reverb engine.

  • Knob 1: Dry and wet balance.
  • Knob 2: Room size.
  • Knob 3: Reverb decay.
  • Knob 4: Unused in this menu.
  • Edit button: Press once to start or stop editing.
Noise

Noise

Add and shape a noise layer alongside the main oscillator.

  • Knob 1: Noise level.
  • Knob 2: Noise filter cutoff.
  • Knob 3: How strongly the envelope shapes the noise.
  • Knob 4: Noise type: none, white, black, brown, pink, or red.
  • Edit button: Press once to start or stop editing.
Harmonist

Harmonist

Create up to three harmony voices and tune how each added voice behaves.

  • Knob 1: Level of the current harmony voice.
  • Knob 2: Number of harmony voices, from 0 to 3.
  • Knob 3: Interval, in semitone steps, for the current harmony voice.
  • Knob 4: Detune for the current harmony voice.
  • Edit button: Press once to start or stop editing.
  • Note: When you add a new harmony voice, that newest voice becomes the one Knob 1, Knob 3, and Knob 4 are editing.
Write

Write

Record a note sequence, then play it back using the sound settings that were active when recording started.

  • Knob 1: Write playback volume.
  • Knob 2: Playback filter tilt: left is more low-pass, center is flat, right is more high-pass.
  • Knob 3: Note hold length.
  • Knob 4: Playback tempo.
  • Edit button, single press from idle: Starts a fresh recording, clears the old Write sequence, and captures the current sound settings for playback.
  • While recording: Play notes to add them to the sequence.
  • Edit button, single press while recording: Stops recording.
  • Edit button, hold while idle: Auditions the sequence while the button is held down.
  • Edit button, triple-click: Starts looping playback.
  • Edit button, single press while looping: Stops the loop.
  • Edit button, double-click: Returns to the main menu.
Presets

Presets

Save, reorder, delete, and load complete sound setups.

  • Knob 2-4 in the list: Move through the preset list.
  • Knob 1 in the list: Reorders the currently selected preset.
  • Any knob in the Delete / Enter screen: Switches between the two options.
  • Edit button on a preset: Opens the Delete / Enter screen.
  • Edit button on Add: Opens preset naming for the current settings.
  • During naming: Use the note keys to enter letters A through W. The X key acts as backspace.
  • Edit button during naming: Saves the new preset.
  • Choosing Enter: Loads the preset and returns to the main menu.
  • Edit button, double-click: Returns to the main menu without loading anything.

In every submenu, double-clicking the Edit button returns to the main menu. The only difference is what a single press does on that screen.
